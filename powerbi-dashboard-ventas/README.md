
# Dashboard de Ventas - Power BI

Este proyecto consiste en un dashboard de análisis de ventas creado en Power BI.

## Dataset
El archivo `dataset_ventas.xlsx` contiene información de ventas con:
- Fecha
- Producto
- Cantidad
- Precio
- Ventas

## Pasos para crear el dashboard

1. Abrir Power BI Desktop
2. Cargar el archivo `dataset_ventas.xlsx`
3. Crear visualizaciones:

- Ventas por mes
- Ventas por producto
- KPI Total de ventas
- Segmentación por fecha

## Objetivo
Mostrar habilidades en visualización de datos y análisis con Power BI.
