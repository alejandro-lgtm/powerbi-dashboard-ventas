
# Dashboard de Ventas - Power BI

Este proyecto consiste en un dashboard de análisis de ventas creado en Power BI.

## Dataset
El archivo `dataset_ventas.xlsx` contiene información de ventas con:
- Fecha
- Producto
- Cantidad
- Precio
- Ventas

## Funcion

- Ventas por mes
- Ventas por producto
- KPI Total de ventas
- Segmentación por fecha

## Objetivo
Mostrar habilidades visualización de datos y análisis con Power BI.
